# Division Of Data

>[!data]
>**Data is divided into 3 parts**

| Dataset Part | Purpose |
|---------------|---------|
| Training Set | Used to feed the model and train it by learning patterns from the data. |
| Validation Set | Used to check if the model performs well on unseen data and to tune model parameters. |
| Test Set | Used for final evaluation to measure the model's real-world performance. |
