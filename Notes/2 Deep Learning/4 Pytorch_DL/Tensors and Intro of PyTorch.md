# Pytorch

- #### Syntax to import: 

```python
import torch
```

# Tensors

- #### Tensor is the core data structure used in deep learning which is used to store and process data.

- #### A **tensor** is simply a multi-dimensional array:
	- **0D tensor** → scalar (e.g., `5`)
	- **1D tensor** → vector (e.g., `[1, 2, 3]`)
	- **2D tensor** → matrix (e.g., a table of numbers)
	- **nD tensor** → higher-dimensional data (e.g., images, videos)
